# How to run

```bash
cargo run -- --username "my_username"
```

# Output example

```
Username: hell
Serial: 0x000004F6
Password: hvdduuuu

Command to execute:
./task09.exe " " -u hell -p hvdduuuu -f FILE_PATH
```

Copy and execute the revealed command.
Keep in mind, that `ARE_YOU_SURE` ENV variable must be set.
